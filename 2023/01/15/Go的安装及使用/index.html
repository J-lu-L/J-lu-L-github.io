Go的安装

一、下载Go

  • 因为官方下载速度过慢，推荐在go语言中文网下载
    [下载地址](Go下载 - Go语言中文网 - Golang中文社区 (studygolang.com))

二、选择适合自己系统的安装包

三、安装Go

  • 如果你的C盘内存足够多一直点击下一步即可，否则建议你换一个安装目录

四、测试是否安装完成

  • win+R打开命令行窗口输入cmd并按回车

  • 输入go version 按回车，出现一下画面即安装成功

五、配置环境变量

  • 在搜索栏搜索环境变量，并选择打开

  • 需要配置GOROOT和GOPATH

  • GOROOT为你Go的安装目录，GOPATH为你的想要存放代码的文件夹（GOPATH默认存在，你可以用默认的也可以自己修改）

  • 像我的GOPATH的地址就是我新建的在D盘的Goworks文件夹
    并且为了后续的开发方便建议在Goworks内建立以下三个文件夹
    （修改GOPATH的时候记得把用户变量一起修改，否则修改不成功）

-使用win+r打开命令行窗口，输出go env测试是否修改成功

Go语言的开发工具的选择

一、如果手头宽裕推荐Goland(因贫穷未能详细教学)

[Goland官网地址](Download GoLand: A Go IDE with extended support for JavaScript, TypeScript, and databases (jetbrains.com))

二、如果手头不太宽宽裕推荐VScode

[VScode官方地址](Visual Studio Code - Code Editing. Redefined)

  • 安装好后打开vscode，在左边导航栏选择红框所示按钮安装go开发插件

  • 在搜索框输入go，并选择第一个插件进行安装

  • 新建一个test.go文件作为测试

**按F5运行，然后会提示要更新插件，但是当你安装的时候会报错说无法安装

  • 解决办法在右下角terminal，输入一下命令并重启VScode即可解决
    go env -w GO111MODULE=on
    go env -w GOPROXY=https://proxy.golang.com.cn,direct

  • 重新打开VScode运行代码，出现提示需要安装，点击install all即可
  • 当全部安装完成之后，再次运行代码，运行成功即配置成功
